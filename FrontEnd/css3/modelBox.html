Modelo de cajas en Css3

El modelo de cajas es la forma en que Css interpreta a los elementos HTML (cajas).

  • Borde (border)
  • Margen (margin)
  • Relleno (padding)

Dentro de este modelado se cuenta con los cuatro lados de la caja.

  • Arriba (top)
  • Abajo (bottom)
  • Izquierda (left)
  • Derecha (right)
  • Ancho (width)
  • Alto (height)

Border

Es lo que delimita cada elemento HTML.

Margin

Son las distancias externas. Es decir las distancias entre el elemento HTML y sus elementos hermanos o padres.

Padding

Son las distancias internas. Es decir la distancia del borde al contenido.

Propiedad display

La propiedad display permite modificar el comportamiento de linea o bloque de una etiqueta, sus valores son inline (linea), block (bloque),inline-block (se comporta como de linea pero acepta modificar sus dimensiones de alto y ancho), none (no se visualiza pero carga en el navegador)
